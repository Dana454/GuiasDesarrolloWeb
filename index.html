Datos generales de la materia

El curso de Desarrollo Web introduce los fundamentos esenciales para construir sitios web modernos y funcionales. Abarca HTML, CSS y JavaScript, junto con principios de maquetación, diseño responsivo y buenas prácticas de programación. Los estudiantes aprenden a estructurar contenido, aplicar estilos visuales, crear interfaces interactivas y comprender la arquitectura de aplicaciones front-end, desarrollando habilidades para implementar soluciones web adaptables y accesibles.

Guía 1

Esta guía es sobre los conceptos básicos de HTML y CSS, enseña cómo estructurar documentos web y aplicar estilos visuales mediante distintos métodos: estilos embebidos, etiquetas y hojas de estilo externas.

Guía 2

Esta guía es sobre selectores CSS avanzados y el modelo de caja, enseña cómo aplicar estilos precisos mediante combinaciones de selectores, pseudo-clases y pseudo-elementos, y cómo estructurar visualmente los elementos usando márgenes, bordes, rellenos y contenido.

  • Ejercicio 1: película
  • Ejercicio 2: tablero de damas
  • Ejercicio 3: biografía

Guía 3

Esta guía es sobre posicionamiento, visualización y maquetación en CSS, enseña cómo organizar y distribuir elementos en la página mediante propiedades como display, position, float y flexbox, facilitando el diseño estructurado de sitios web.
